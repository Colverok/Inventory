**Прототип простой системы инвентаря с drag & drop.**

**Инвентарь:**

- Ячейки с фиксированным количеством слотов.
  
- Поддержка нескольких типов предметов (например: оружие, зелье, квест-предмет).
  
- Предметы могут быть "стекамыми" (например, зелья) и не стекаемыми (например, оружие).
  
**Предметы:**

У каждого предмета есть:

-название;

-иконка;

-описание;

-тип (enum);

Реализованы через ScriptableObject.

**Функционал:**

-Перетаскивание предметов между слотами (drag & drop).

-Использование предмета по двойному клику / по кнопке "Use".

-Удаление предмета (кнопка "Delete").

-Вывод информации о предмете при наведении (tooltip с описанием).

-Сортировка предметов по типу / имени / количеству.

-Сохранение и загрузка инвентаря (PlayerPrefs + JSON).
